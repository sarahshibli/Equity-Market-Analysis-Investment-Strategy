# Equity Research & Portfolio Allocation Project

## Overview
This project focuses on analyzing the financial performance of major technology companies—Apple, Microsoft, and Nvidia—compared to the S&P 500 index. The goal is to apply data analysis and financial concepts to evaluate company performance and support investment decision-making.

## Objective
- Compare the financial and market performance of selected companies
- Analyze trends in growth, profitability, and stock returns
- Evaluate risk vs. return characteristics
- Develop a data-driven portfolio allocation strategy

## Companies Analyzed
- Apple
- Microsoft
- Nvidia

## Benchmark
- S&P 500 Index

## Tools & Technologies
- Microsoft Excel
- Python
- Tableau

## Planned Analysis
The project will involve:
- Collecting and cleaning financial and stock data
- Analyzing key financial metrics such as revenue, profit margins, and stock performance
- Comparing company performance against the S&P 500
- Creating visualizations to highlight trends and insights

## Expected Outcome
- Identification of differences in company performance
- Insights into growth and risk characteristics
- A final portfolio allocation strategy based on data analysis
